# Mac Cleaner

macOS에서 불필요한 앱 잔여물과 일반 캐시를 정리하는 작은 CLI 도구입니다.

## 용도

이 도구는 다음 같은 항목을 정리하는 데 초점을 맞춥니다.

- 삭제된 앱의 잔여 폴더
- 사용자 캐시와 로그
- 개발 도구 캐시
- Homebrew 관련 캐시

반대로, macOS가 보호하는 메타 파일이나 시스템성 항목은 제외합니다.

## 실행 방법

### 1. 기본 실행

```bash
python3 mac_cleaner.py
```

기본적으로는 정리 가능한 항목을 자동으로 선택하고, 마지막에 한 번 확인합니다.

### 2. 바로 정리

```bash
python3 mac_cleaner.py --yes
```

최종 확인 없이 바로 정리합니다.

### 3. 직접 선택

```bash
python3 mac_cleaner.py --interactive
```

번호를 직접 골라서 정리할 수 있습니다.

### 4. macOS 런처

```bash
./run_mac_cleaner.command
```

터미널에서 바로 실행하기 편한 런처입니다.

### 5. 더블클릭 앱

Finder에서 아이콘처럼 더블클릭해서 실행하려면 아래 앱을 사용하세요.

```text
앱클리너.app
```

앱은 `앱클리너.app` 안에서 GUI 창으로 실행됩니다.

GUI에서는 스캔 버튼만 누르면 결과를 보여주고, 계속 진행할지 마우스로 선택할 수 있습니다.

## 설치

이 프로젝트는 외부 패키지 없이 표준 라이브러리만 사용합니다.

필요하면 가상환경을 만들어서 실행할 수 있습니다.

```bash
python3 -m venv .venv
source .venv/bin/activate
python3 mac_cleaner.py
```

## 제외되는 항목

다음 계열은 기본적으로 건너뜁니다.

- macOS 보호 메타 파일
- 시스템/예약 이름
- 권한상 현재 사용자로 수정할 수 없는 항목

## 주의

이 도구는 삭제 작업을 수행하므로, 실행 전에 중요한 파일이 있는지 한 번 확인하는 것이 좋습니다.
